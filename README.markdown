# FireLogger Addon

FireLogger is a sexy logger console integrated into [Firebug][firebug]. You'll need Firebug 1.2 or higher.

## Supported Languages

  * **[FirePython][firepython]** (production ready)
  * **[FireRuby][fireruby]** (in planning)
  * add your own here
